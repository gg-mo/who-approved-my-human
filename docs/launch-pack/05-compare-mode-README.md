# 05 - Compare Mode (Cross-Agent)

## v1.1 Feature

Allow users to run the same test with multiple agent outputs and compare type results.

Example:

- Claude: `CKVD`
- ChatGPT: `CKTD`
- Coding Agent: `CKVH`

## Scope

- Keep this lightweight at first
- Compare cards side-by-side
- Highlight where letters match and differ

## UX Principle

Frame differences as perspective, not correctness.

Suggested copy:

- "Different agents notice different sides of you."

## Retention Impact

This is a built-in replay loop and gives users a reason to run the test multiple times.
