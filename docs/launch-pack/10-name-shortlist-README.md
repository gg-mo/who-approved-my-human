# 10 - Name Shortlist

## Recommendation

**Best pick: `HumanGrade`**

Why:

- Short and brandable
- Instantly implies "AI grading humans"
- Fun but not cringe
- Works for both meme mode and deeper reflection mode later

## Strong Alternatives

- `AgentMirror`
- `HowMyAISeesMe`
- `Promptality`
- `HumanType`
- `JudgedByAI`

## More Playful Alternatives

- `ReadMeRobot`
- `VibeCheckByAI`
- `MyAIsOpinion`
- `Type Me, Bot`

## Name Direction Notes

- Avoid names that sound overly corporate or clinical
- Avoid names that are too harsh (feels like punishment)
- Favor names that can sit on a share card naturally

## Landing Tagline Pairing (for HumanGrade)

- Headline: "Your AI has opinions about you."
- Subhead: "Get your agent-eye personality type in 2 minutes."
